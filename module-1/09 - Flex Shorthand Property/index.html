More Properties for Flex Items

flex-shrink, flex-grow, and flex-basis

flex-shrink determines how much a flex item will shrink relative to the other items in the flex container when there is not enough space. By default, flex-shrinkflex-shrink is set to 1, which means that all items will shrink proportionally if necessary. If you set flex-shrink to 0, the item will not shrink at all, regardless of the available space. This property is particularly useful when you want to ensure that certain elements maintain their size, even if it means other elements have to shrink more significantly.

flex-grow is used to define how much a flex item will grow relative to the other items in the flex container when there is extra space. The default value for flex-grow is 0, meaning items will not grow by default. However, by setting flex-grow to a positive integer, you can make items expand to fill the available space. For example, if you set flex-grow to 1, the item will grow to occupy the available space proportionally with other flex items. Setting flex-grow to a higher value will make the item grow more in relation to items with lower values.

flex-basis specifies the initial main size of a flex item before any extra space is distributed according to the flex-grow and flex-shrink factors. The default value is auto, meaning the item's size is based on its content. However, you can set flex-basis to a specific length, such as 20% or 100px, to determine the item's initial size explicitly. This property is useful for establishing a baseline size for flex items before they are adjusted for space distribution.

The flex Shorthand

The flex shorthand property combines flex-grow, flex-shrink, and flex-basis into a single declaration, making it easier to set all three properties at once.

By default, flex is set to 0 1 auto, which means the item will not grow, will shrink if necessary, and its initial size is determined by its content. The shorthand can take one, two, or three values. When a single value is provided, it applies to flex-grow. If two values are provided, they apply to flex-grow and flex-shrink. Providing three values specifies all three properties explicitly.
